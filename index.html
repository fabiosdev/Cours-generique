  • ANNEXE : Définitions

    1. Vocabulaire générique :
      1. FE / BE : (développement front end / back end)
      2. HTTP : HyperText Transfer Protocol
      3. HTTPS : HyperText Transfer Protocol Secure
      4. USERSTORIES : Phases de développement : Fonctionnalités / production
      5. BACKLOG : Classement des fonctionnalités (production Back) doit être étudié ; trié ci nécessaire avant retour client
      6. CMS : Content Management System
      7. FRAMEWORK: désigne un ensemble cohérent de composants logiciels structurels, qui sert à créer les fondations ainsi que les grandes lignes de tout ou d’une partie d'un programme / site.
      8. SEO : Search Engine Optimization : (référencement gratuit)Ensemble de techniques visant à optimiser la visibilité d'un page web dans les pages de résultats de recherche (les SERP, pour Search Engine Result Pages). Ces techniques cherchent à améliorer la compréhension par les robots d'indexation de la thématique et du contenu d'une ou de l'ensemble des pages d'un site web et à en augmenter ainsi le trafic naturel.
      9. SEA : ( Search Engine Advertising) : Le référencement payant, en anglais (en)Search Engine Advertising (SEA), est une discipline proche du SEO et du Search Engine Marketing (SEM), qui lui sont généralement associées.
      10. UX Design : (User eXperience Design) : Test Utilisateur / Analyse / Fonctionnalités / Architecture du site / Satisfaction client / Interactions Homme – Machine / Conception / Prototypage / Collaboration.
      11. UI Design : (User Interface Design) : Identité visuelle / Design graphique / Charte éditoriale)
      12. SCRUM : Framework de développement et de maintenance (méthode Agile)
      13. AGILE : Les méthodes agiles se veulent plus pragmatiques que les méthodes traditionnelles. Elles impliquent au maximum le demandeur (client) et permettent une grande réactivité à ses demandes.
      14. FLAT DESIGN : Design « Plat »
      15. RESPONSIVE DESIGN : site web adaptatif (anglais RWD pour responsive web design, conception de sites web adaptatifs)
